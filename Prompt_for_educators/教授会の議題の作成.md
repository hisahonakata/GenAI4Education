# 教授会の議題の作成

# プロンプト例

```
あなたは経験豊富で洞察力に満ちた学校管理者であり、効率的かつ生産的な会議の計画と実施に関する深い知識を持っています。{実施間隔}開催される{会議時間}の教授会の議題を作成する任務が与えられています。この会議では、以下の要素を含める必要があります：

1. イントロダクション: 会議の目的と参加者の紹介を含む、簡潔ながらも効果的な開始部分。

2. 管理者からのアップデート: カリキュラムディレクター、アドミッションディレクター、テクノロジーディレクターそれぞれからの最新情報と進捗報告。

3. リマインダーとハウスキーピング: 近日中のイベント、重要な期限、教室の使用規則、健康および安全に関するポリシーの更新など、教職員が注意すべき事項。

4. {特定の課題}

5. クロージング: 会議の主要ポイントのまとめと、次回の会議の予告。

この議題は、時間を最大限に活用し、参加者が情報を共有し、意見を交換するための構造を提供します。また、各セッションの時間割り当てにも注意が必要です。

{実施間隔}:
隔週

{会議時間}:
60分

{特定の課題}:
生成AIに関する新しい単元の作成について
```

# 回答例

効率的な会議計画は、時間の有効活用と参加者間の意見交換を促進する鍵です。以下は、隔週で開催される60分の教授会の議題案です。

### 1. イントロダクション (5分)
- 目的の明確化: 会議の主目的と目標を説明。
- 参加者の紹介: 簡潔に各参加者の役割と名前を紹介。

### 2. 管理者からのアップデート (15分)
- カリキュラムディレクター: 最新のカリキュラム変更点や進捗。
- アドミッションディレクター: 入学プロセスのアップデートや統計データ。
- テクノロジーディレクター: 教育技術の最新動向や導入計画。

### 3. リマインダーとハウスキーピング (10分)
- 近日中のイベントや期限の確認。
- 教室の使用規則、健康および安全に関するポリシーの更新。

### 4. 特定の課題: 生成AIに関する新しい単元の作成 (20分)
- 新単元の目的と期待される成果の提示。
- アイデア共有とディスカッション: 教員からの意見や提案を収集。

### 5. クロージング (10分)
- 会議の主要ポイントの要約。
- 次回会議の日程と予定されるトピックの予告。

この議題は、時間管理に重点を置きつつ、全員が情報を共有し、活発な議論を行えるよう設計されています。会議の流れをスムーズに進行させ、参加者が積極的に参加する環境を作ることが重要です。
